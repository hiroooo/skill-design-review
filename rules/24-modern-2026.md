# 「2026 年 確実に古い」10 連発

これが見つかったら問答無用で priority `high` か `medium`。レポート冒頭の警告枠に出す。

## 1. Comic Sans / Papyrus / Impact 直使い
- 30 年変わらない「素人」記号
- → `high`

## 2. 完全黒 (`#000000`) / 完全白 (`#ffffff`) のみ
- 2026 では gray-900 / gray-50 の方が標準
- → `medium`

## 3. 線形 easing (linear) 多用
- アニメーションが物理的に不自然
- → `medium`、cubic-bezier 推奨

## 4. mobile で max-width 未設定 (margin: 0 端張り付き)
- 2026 は max-width: min(90vw, 1200px) が標準
- → `high`

## 5. グラデーションボタン (top→bottom 白→灰)
- 2000 年代 Web style、Bootstrap 2 系
- 2026 は flat button + subtle shadow
- → `high`

## 6. font-weight 1-2 種だけ
- 階層感ゼロ、フラットすぎる
- 2026 は variable font で 4-6 段階 weight が普通
- → `medium`

## 7. border-radius 4 種以上混在
- design token 不在の証拠
- 2026 は 2-3 種で統一
- → `medium`

## 8. section margin 全部同じ (リズム破壊)
- 階層感ゼロ、流し読みできない
- 2026 は 64/96/128px のような階層的 rhythm
- → `medium`

## 9. hero に visual なし (文字だけ)
- 2026 SaaS LP の標準は「hero に視覚」
- Apple / Vercel / Stripe / Linear / Figma すべて hero img あり
- → `medium` (Editorial 狙いなら friendly)

## 10. box-shadow opacity > 0.5 + blur < 10
- ハードな黒 shadow、2010 年代 Material Design v1 風
- 2026 は rgba(0,0,0,0.05-0.12) の極薄
- → `low` (頻発なら medium に格上げ)

## ボーナス: 2026 年「これは新しい」シグナル

逆に **見つけたら積極的に褒める** 良い点:

1. **variable font** (`font-variation-settings`) 使用
2. **container queries** (`@container`) でレスポンシブ
3. **CSS color-mix()** / `oklch()` 系 wide gamut
4. **subtle gradient mesh** (Vercel / Stripe 風)
5. **scroll-driven animation** (`animation-timeline: scroll()`)
6. **view-transition** API
7. **iOS Liquid Glass** 風 (`.ultraThinMaterial`, depth layering)
8. **dynamic color** (Material You) on Android
9. **Inter / Geist / Pretendard / Söhne** などの 2026 スタンダード
10. **subtle micro-interaction** (haptic feedback, springy scroll)
